style="color:#008000" href="https://curseforge.com/minecraft/modpacks/the-vanilla-experience" rel="nofollow">The Vanilla Experience</a></span>.</strong><br><span style="font-size:18px">Advancement Screenshot is a client-side mod which can automatically document the journey through a world without having to remember to take screenshots manually. Every time an advancement is achieved, a screenshot will be taken. The same code as the screenshot key (F2) is used, so the screenshots are located in the default folder. Works in multiplayer environments with the mod only on the client.<br><br>There is a 1 second delay in between getting the advancement and the actual screenshot. This allows some time for the message in the top right to appear.<br></span><br><br><span style="font-size:24px"><strong><span style="font-size:20px">Configurable:</span> <span style="color:#008000;font-size:14px"><a style="color:#008000" href="https://serilum.com/url/issue-trackerwiki/how-to-configure-mods" rel="nofollow">(&nbsp;how do I configure?&nbsp;)</a></span></strong></span><strong><br></strong><span style="font-size:12px"><strong>showScreenshotTakenMessage</strong>&nbsp;(default = false): If enabled, shows the normal screenshot taken (same as F2) message when a screenshot is taken after an advancement.<br></span><strong>takeScreenshotTickDelay</strong> (default = 20, min 0, max 100): How many ticks the mod should wait until it takes a screenshot.
